## Week-by-Week Action Plan (Feb-May 2026)

### 🎯 DUAL-TRACK STRATEGY
*Run Navy NUPOC and civilian applications in parallel until you have offers to compare.*

### 📅 FEBRUARY 2026 - FULL SPRINT

**Week 1 (Feb 1-7): Foundation**

*Civilian Track:*
- [ ] Update resume with all recommendations below
- [ ] Complete LinkedIn profile optimization (see section below)
- [ ] Contact BAE Systems former supervisor for referral
- [ ] Apply to 5 Tier 1 companies
- [ ] Create spreadsheet to track all applications

*Navy NUPOC Track:*
- [ ] **Find Navy Officer Recruiter** (Google "Navy Officer Recruiter Maryland" or navy.com)
- [ ] **Make initial contact** - express interest in NUPOC program
- [ ] **Request NUPOC information packet and study guide**
- [ ] Order official transcripts for Navy application

**Week 2 (Feb 8-14): Expand Applications**

*Civilian Track:*
- [ ] Apply to 10+ positions across all tiers
- [ ] Start capstone portfolio/slide deck
- [ ] Schedule UMD Career Fair appointments if available
- [ ] Research companies' interview processes

*Navy NUPOC Track:*
- [ ] **Submit transcripts to recruiter** for eligibility review
- [ ] **Begin application paperwork** (SF-86 security form, etc.)
- [ ] Start reviewing calculus and physics fundamentals
- [ ] Schedule medical screening if recruiter advises

**Week 3 (Feb 15-21): Network Push**

*Civilian Track:*
- [ ] Message 10 UMD alumni working at target companies (LinkedIn)
- [ ] Join relevant LinkedIn groups (ASME, SAE, industry-specific)
- [ ] Apply to 10+ more positions
- [ ] Practice STAR method for behavioral interviews

*Navy NUPOC Track:*
- [ ] **Complete any remaining paperwork**
- [ ] Continue studying NUPOC study guide
- [ ] Prepare "Why Navy? Why Nuclear?" talking points
- [ ] Ask recruiter about VIP orientation trip (tour submarine/ship)

**Week 4 (Feb 22-28): Technical Prep**

*Civilian Track:*
- [ ] Review FEA fundamentals—common interview topic
- [ ] Brush up on SolidWorks—may have practical tests
- [ ] Continue applications (maintain 15+/week)
- [ ] Follow up on any recruiter contacts

*Navy NUPOC Track:*
- [ ] Continue NUPOC study guide review (calculus, physics, thermo)
- [ ] Practice explaining technical concepts out loud
- [ ] Prepare leadership examples for interviews

### 📅 MARCH 2026 - PEAK INTERVIEW SEASON

**Week 1-2 (Mar 1-14)**

*Civilian Track:*
- [ ] Apply to 20+ more positions
- [ ] Prepare for technical interviews (design problems, FEA questions)
- [ ] Practice explaining capstone project in 2 minutes
- [ ] Begin mock interviews with Career Center

*Navy NUPOC Track:*
- [ ] **Complete pre-screening** (medical, security)
- [ ] **Schedule DC interview** (target late March or April)
- [ ] Intensify technical study (calculus, physics, engineering)
- [ ] Practice problem-solving under pressure

**Week 3-4 (Mar 15-31)**

*Civilian Track:*
- [ ] Follow up on applications submitted 2+ weeks ago
- [ ] Continue applications—cast wide net
- [ ] Network at any available career events
- [ ] Research salary negotiation tactics

*Navy NUPOC Track:*
- [ ] **Final interview prep** - review all technical fundamentals
- [ ] Prepare "Why Navy Nuclear?" story
- [ ] Review your transcript—they'll ask about specific courses
- [ ] Get good sleep, be physically ready

### 📅 APRIL 2026 - OFFERS & DECISIONS

**Week 1-2 (Apr 1-14)**

*Civilian Track:*
- [ ] Follow up aggressively on pending applications
- [ ] Continue interviewing
- [ ] Evaluate any offers received
- [ ] Negotiate if appropriate (always counter!)

*Navy NUPOC Track:*
- [ ] **NAVAL REACTORS INTERVIEW IN DC** 🎯
- [ ] Give your absolute best—this is the filter
- [ ] Wait for selection decision (usually within 2 weeks)

**Week 3-4 (Apr 15-30)**

*Both Tracks:*
- [ ] **Compare all offers** (civilian + Navy if selected)
- [ ] Decision time—weigh tradeoffs carefully
- [ ] Continue applying if no offers yet (don't panic)
- [ ] Discuss options with family, mentors
- [ ] Finalize any remaining coursework

### 📅 MAY 2026 - TRANSITION

**Week 1-2 (May 1-14)**
- [ ] **Make final decision** on career path
- [ ] Accept offer (civilian OR Navy)
- [ ] If Navy: Enlist, begin receiving NUPOC pay immediately
- [ ] If civilian: Complete onboarding paperwork
- [ ] Graduation! 🎓

**Week 3-4 (May 15-31)**

*If Civilian Path:*
- [ ] Relocation planning if needed
- [ ] Start date preparation
- [ ] Thank your network/references

*If Navy NUPOC Path:*
- [ ] Enjoy final weeks before OCS
- [ ] Physical fitness preparation
- [ ] Report to OCS (Newport, RI) - typically June/July/August

### 📊 Application Targets (Civilian Track)
- **February**: 40+ applications
- **March**: 50+ applications
- **April**: 30+ applications (or as needed)
- **Total goal**: 100-150 applications

*Yes, this is a lot. The market is competitive. Treat job searching as a part-time job (15-20 hours/week).*
