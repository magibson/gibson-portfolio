## JARVIS Dashboard - Agent Status Tracking

**IMPORTANT:** When spawning or completing subagents, update the agent status file:
- File: `/home/clawd/clawd/projects/morning-dashboard/agent-status.json`
- Format:
```json
{
  "main": {"name": "JARVIS", "status": "online", "lastActive": "ISO_DATE"},
  "agents": [
    {"label": "task-name", "status": "running", "task": "Description", "startedAt": "ISO_DATE"}
  ]
}
```
- When spawning: Add to agents array with status "running"
- When complete: Remove from agents array (or set status to "completed")
- The fetch_data.py script reads this and includes it in data.json every 5 minutes
- Dashboard displays this in the agent network visualization
