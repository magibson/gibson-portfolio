## DETAILED BREAKDOWN

### Propwire (FREE)
- 157M+ properties nationwide
- Filter by: MLS Sold, High Equity, Absentee, Pre-foreclosure, etc.
- Includes owner names and mailing addresses
- Up to 10,000 exports/day
- No API, but scrapable
- **Perfect for getting the raw property data**

### Tracerfy ($0.02/record)
- Skip tracing only (needs name + address input)
- API available with Bearer token auth
- No monthly minimum
- 70-90% accuracy
- Webhook support for automation
- **Perfect for converting addresses to phone numbers**

### DealMachine ($99/mo)
- Unlimited skip tracing included
- Property data with owner names
- List Builder with 700+ data points, 70+ filters
- "MLS Expired", equity filters, etc.
- API + Zapier integration
- Route tracking, direct mail, CRM built-in
- **Best all-in-one for someone who wants simple**

### PropStream ($99-149/mo)
- 160M+ property records
- Extensive filters including recent sales
- Skip tracing $0.12/record (free on Pro or with Lead Automator add-on)
- API available
- Industry standard for real estate investors
- **Most comprehensive data but more complex**

### BatchLeads ($99/mo Growth)
- 10,000 leads/month with contact info included
- Property search with distress filters
- Skip tracing counted in lead quota
- Zapier integration
- Direct mail, SMS integration
- **Good middle ground**
