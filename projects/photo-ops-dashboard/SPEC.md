# Local Photo Ops Dashboard

A simple single-page dashboard for Matt to check local photography conditions.

## Features
1. **Current Weather** - Temperature, conditions, wind for Shrewsbury NJ
2. **Golden Hour Times** - Sunrise, sunset, golden hour windows for today
3. **Shooting Conditions** - Simple "Good/Fair/Poor" rating based on weather
4. **Quick Locations** - Links to winter photo spots with drive times
5. **Recent Alerts** - Placeholder for urgent photo ops (wildlife sightings, weather events)

## Tech
- Single HTML file with embedded CSS/JS
- Use wttr.in for weather (no API key needed)
- Calculate golden hour from sunrise/sunset
- Mobile-friendly design
- Dark mode for early morning use

## Data
Location: Shrewsbury, NJ (40.3268, -74.0618)
Winter locations from: ~/clawd/projects/winter-photo-spots-2026-02-02.md
